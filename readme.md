# LuckSweeper

論理で解ける部分はBotが自動処理し、運が必要な場面だけプレイヤーに回すマインスイーパー。

![Python](https://img.shields.io/badge/Python-3.11+-blue.svg)
![PySide6](https://img.shields.io/badge/GUI-PySide6-green.svg)

## 機能

- **3段階ソルバー** — 基本制約 → サブセット解析 → CSPバックトラッキング
- **ヒント矢印** — 手詰まり時に最も安全なセルを▼で示し、地雷確率%を表示
- **並列処理** — ボットの思考を別スレッドで実行、UIフリーズなし
- **初手安全保証** — 最初のクリックとその周囲8マスに地雷がないことを保証
- **ズーム / パン** — ホイールで拡大縮小（カーソル中心）、右/中クリックドラッグで移動
- **10テーマ** — Modern / Sea / Classic / Brazil / Hoover / Retro / Sakura / Arctic / Sunset / Hacker
- **日英対応**

## ソルバーの仕組み

1. **基本制約** — 数字 = 周囲フラグ数 → 残りは安全 / 数字 = フラグ + 未開放数 → 残りは地雷
2. **サブセット解析** — 制約Aのセル集合が制約Bの部分集合なら、差分から地雷/安全を推論
3. **CSP** — フロンティアセルの全配置をバックトラッキングで列挙し、全配置で地雷/安全なセルを確定
4. **ヒント** — CSPの確率情報から最も安全なセルをプレイヤーに提示

## 使い方

```bash
pip install PySide6
python mine.py
```

- 左クリックでセルを開く
- Botが論理的に解ける部分を自動処理
- 手詰まり（運の場面）で▼矢印がヒントを表示
- ホイールで拡大縮小、右/中クリックドラッグで移動
- ゲームオーバー後クリックでリスタート
